## Implementation Details

### 1. Create `audit_report.py`

A standalone module that generates the PDF using ReportLab. Use `create_pdf_handout.py` as a style reference for Calibrium branding (same colors, fonts, header/footer pattern).

**Report structure (pages):**

**Page 1: Cover Page**
- Calibrium logo (white version on navy background)
- "Regulatory Audit Report"
- "Project Sentinel -- Treasury Operations"
- Report period (date range parameter)
- Generation timestamp
- "CONFIDENTIAL" watermark

**Page 2: Executive Summary**
- Report period
- Total capital calls processed (count)
- Breakdown: Executed / Rejected / Escalated (with counts and EUR totals)
- Total EUR executed in period
- Number of unique reviewers
- Number of wire changes in period
- Number of commitment amendments in period
- Key risk events (any wire mismatches or over-commitments)

**Page 3: Commitment Tracker Snapshot**
- Full commitment tracker table as of report generation date
- Total commitment, total funded, total remaining
- % deployed

**Pages 4+: Processed Capital Calls Detail**
- One table entry per processed call, showing:
  - Timestamp, filename, fund name (extracted + matched), match score
  - Amount, currency, due date
  - Commitment check result + message
  - Wire check result + message
  - Overall status, action taken
  - Reviewer name
  - Review notes (if any)
- Color-code rows: green background for EXECUTED, red for REJECTED, amber for ESCALATED

**Wire Changes Section (if any):**
- Table of all wire change requests in the period
- Old value, new value, reason, requested by, reviewed by, status

**Commitment Amendments Section (if any):**
- Table of all amendments in the period
- Fund, increase amount, reason, requested by, reviewed by, status

**Final Page: User Activity Summary**
- Table showing each user, their role, and how many calls they processed/reviewed
- Certification line: "This report was auto-generated by Project Sentinel"

### 3. Add button to Audit Log page in `app.py`

On the Audit Log page, after the filter controls and before the card display, add:

```python
# Audit Report generation
with st.expander("Generate Regulatory Audit Report"):
    report_col1, report_col2 = st.columns(2)
    with report_col1:
        report_date_from = st.date_input("From", value=datetime(2026, 1, 1), key="report_from")
    with report_col2:
        report_date_to = st.date_input("To", value=datetime.now(), key="report_to")
    
    if st.button("Generate PDF Report", type="primary"):
        with st.spinner("Generating audit report..."):
            from audit_report import generate_audit_report
            pdf_bytes = generate_audit_report(
                date_from=report_date_from.strftime("%Y-%m-%d"),
                date_to=report_date_to.strftime("%Y-%m-%d"),
            )
        st.download_button(
            "Download Audit Report PDF",
            data=pdf_bytes,
            file_name=f"sentinel_audit_report_{report_date_from.strftime('%Y%m%d')}_{report_date_to.strftime('%Y%m%d')}.pdf",
            mime="application/pdf",
        )
```
